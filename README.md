# ROS Camera DATA 연결

### 패키지 설치
$sudo apt-get install ros-melodic-uvc-camera 
$sudo apt-get install ros-melodic-image-*
$sudo apt-get install ros-melodic-imate-view


### 터미널 각각 입력 
$roscore 
$rosrun uvc_camera uvc_camera_node
$rqt_image_view
